# Repository Signing Key

The public key in this directory verifies Daoliyu APT and RPM repository
metadata. Confirm its fingerprint through an independent official Daoliyu
channel before trusting it.

```text
B218 D900 62A6 E2B4 1389 47B0 1494 FE30 ABD9 F22F
```

The primary Ed25519 certification key expires on 2031-07-13. Its current
Ed25519 repository-signing subkey expires on 2028-07-13 and can be rotated
without replacing the trusted primary key.
