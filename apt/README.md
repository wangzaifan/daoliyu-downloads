# APT Beta Repository

Repository root: `https://repo.daoliyu.cn/apt`

Suite: `beta`

Component: `main`

Architectures: `amd64`, `arm64`

Package payloads are stored as immutable GitHub Release assets. Signed package
indexes use versioned paths such as `pool/beta/v0.3.4-beta/<package>.deb`; ESA
maps those paths to the matching release tag without using the API origin.
